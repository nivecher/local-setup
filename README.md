# local-setup

Local setup for development for various platforms

## Instructions

### Base Setup

Update O/S packages, setup languages, etc.

```
sudo ./base-setup.sh
```

### Tools Setup

Install / update developer tools on the system.  Includes:

* AWS CLI

```
sudo ./tools-setup.sh
```

### User Setup

User specific setup (executed as user,  not system)

```
./user-setup.sh
```
